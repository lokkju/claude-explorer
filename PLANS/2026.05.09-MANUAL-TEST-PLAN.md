## §2 — Issue #1: Connection dialog UX (2 min)

Three bundled sub-fixes. Reproduce by stopping the backend.

```bash
# In the terminal running uvicorn, hit Ctrl+C.
```

Now reload the browser tab. Watch the dialog carefully:

1. **No instant flash:** the dialog should NOT appear immediately. A
   brief connection failure (≤4s) should not surface UI at all.
2. **Spinner-only at retry 2:** at ~5-10 seconds the dialog opens.
   You see "Connecting to Backend" with a spinning icon and "Attempt 2
   of 5". You should NOT see a red "Last error: Failed to fetch" line.
   That was the UGH.
3. **Error appears after terminal failure:** wait ~42s for all 5
   retries to exhaust. The dialog now says "Connection Failed" and the
   red "Last error: …" line appears underneath. (This is correct —
   we WANT the error visible once we've stopped trying.)

Restart the backend:

```bash
DYLD_LIBRARY_PATH=/opt/homebrew/lib uv run uvicorn backend.main:app --reload --port 8000 &
```

Click **Try Again**. The dialog briefly shows "Connecting to Backend"
again, then closes when the backend responds.

**Pass:** all 3 behaviors above. The UGH is gone.

## §3 — Issue #2: Search auto-focus first match (2 min)

New UX: when search results land, the conversation panel auto-scrolls
to the first match and applies a yellow ring (no Cmd+G needed).

1. Open any conversation. Press Cmd+K to open the search panel.
2. Type a query that has matches (e.g., a word you know appears in
   multiple messages of an open conversation). Wait for the result list
   to populate.
3. **Without clicking anything**, look at the conversation panel:
   the first matching message should be in view AND have a yellow
   ring around it. The ring fades after ~2 seconds.
4. Press Cmd+G. The conversation panel scrolls to match 2 (also with
   the yellow ring).
5. Type a different query. The panel auto-scrolls to that query's
   first match.

**Edge case to verify (don't yank back):**

6. Type a query with 2+ matches. Wait for auto-focus.
7. Press Cmd+G to advance to match 2.
8. **Wait 3 seconds without doing anything.** The panel must NOT
   snap back to match 1. The auto-focus is "once per stable-query
   cycle"; subsequent results-refreshes don't yank.

**Pass:** all 5 plus the don't-yank check.
